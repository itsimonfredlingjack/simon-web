# Evidence summary for website content

This file summarizes what the research package supports. It is written for website content work, not as a full research report.

## What can be safely said

- There is a verified publication register with 335 rows.
- 332 rows in the main register have a found original URL.
- The primary original source is `svt.se`.
- A supplemental SVT search produced 56 candidate rows.
- 55 supplemental rows are safe new SVT publications/credits.
- 1 supplemental row is uncertain and should not be counted as confirmed.
- 15 found rows were excluded as already-known or likely duplicate matches.
- The public wording `350+ publiceringar/krediter` is supported as a conservative rounded statement when framed as register plus supplemental SVT findings.

## What the site should emphasize

- Verifierbara arbetsprov.
- Publiceringsvolym med tydlig källstatus.
- Konkreta exempel från SVT.
- Källhantering, dubblettkontroll, and original-URL verification.
- Ability to move from messy events and public information into clear output.

## What the site should not imply

- Do not imply a complete public archive.
- Do not imply career analysis or biographical interpretation.
- Do not imply project ownership or management responsibility where the source only shows byline/credit.
- Do not imply specialist authority in medicine, policing, schools, or AI from publication credits alone.
- Do not treat University of Gothenburg communicator credits as SVT reporting proof.

Use these as compact proof points, not as a table-heavy section.

## Suggested homepage framing

Short version:

```text
Ett källbelagt urval av publiceringar och arbetsprov från SVT, byggt på verifierade originalkällor snarare än lös CV-text.
```

Alternative:

```text
Bakom rubrikerna finns ett verifierat register: hundratals publiceringar, original-URL:er, dubblettkontroll och konkreta exempel på arbete i offentlig informationsmiljö.
```

## Suggested publications page framing

```text
Fältanteckningarna nedan är ett urval. De är inte ett komplett arkiv, utan sex konkreta exempel som visar olika typer av informationsarbete: data, kris, källkritik, fältarbete, organisation och researchdisciplin.
```

## Source hierarchy

Use source weight in this order:

1. Original SVT page with matching title/date/credit.
2. Original GU page with matching title/date/credit, only for GU-specific knowledge-work examples.
3. Main publication register row.
4. Supplemental candidate register row.
5. Search log / excluded matches only as audit trail.
6. Snippet-only or uncertain candidate as uncertainty, not proof.

## Recommended public wording

Use:

- `350+ publiceringar/krediter`
- `verifierade originalkällor`
- `källbelagt urval`
- `arbetsprov från SVT`
- `dokumenterad publiceringsvolym`
- `urval ur verifierat register`

Avoid:

- `komplett arkiv`
- `alla publiceringar`
- `karriärprofil`
- `bevisar expertstatus`
- `ledde arbetet`
- `ägde processen`
- `ansvarade för resultatet`
